# Rascunho de Ideias - Brainstorming Melhorador de Prompts AI

## Técnica 1: Inversão de Suposições (User Choice: IA sugere prompt inicial)

- **Ideia 1.1: "Gerador de Sementes"**: O usuário digita apenas "Marketing para Clínica" e o site gera 3 estruturas de prompt diferentes (Narrativa, Técnico, Criativo) para ele escolher e então "melhorar".
- **Ideia 1.2: "Modo Entrevista" (SELECIONADA)**: A IA faz perguntas rápidas (Tom, Público, Canal) para construir o prompt mestre a partir de respostas curtas.
- **Ideia 1.4: Interface Híbrida**: Mix de botões (para seleções rápidas de modelo e objetivo) com campo de texto livre para o "core" do pedido, permitindo que a IA faça o "bridge" entre o que o usuário quer e o que a máquina entende.

## Técnica 2: Role Playing (Público Amplo)

- **Insights**: O produto deve atender tanto profissionais (Social Media, Redatores) quanto estudantes e usuários recreativos.
- **Estratégia**: O "Modo Entrevista" deve ser dinâmico. Se o usuário escolhe "Imagem", as perguntas mudam para estilos artísticos; se escolhe "Estudo", mudam para profundidade teórica/resumos.

## Técnica 3: SCAMPER (P - Propor para outros usos)

- **Ideia Principal (Biblioteca de Prompts)**: O site oferecerá uma biblioteca curada de prompts prontos e testados para categorias comuns (Marketing, Estudo, Arte). Isso serve como ponto de partida rápido para o assinante.
- **DNA do Produto: Simplicidade e Rapidez**: O diferencial competitivo é a baixa fricção. O usuário entra, escolhe o que quer, melhora o prompt e sai com o resultado pronto para usar na IA de preferência. Sem burocracia.
